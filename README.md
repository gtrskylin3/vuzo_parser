VUZOPARSER - Бот, который поможет отслеживать свое место в конкурсном списке по каждому направлению каждого выбранного вами ВУЗА и будет отправлять уведомление если ваше место в списках изменится

[VUZOPARSR](icon.png)

# Доступные вузы: НГУ; НГТУ НЭТИ; ТГУ 

# Как это работает

### Пользователь может добавить 5 вузов и 5 ссылок на направления в нём
- Пользователь: /start
- Бот: Введите свой код, по которому можно отследить позицию в конкурсе
- Пользователь: 1234567
- Бот: Выберите Вуз: [Кнопка: НГУ] [Кнопка: НГТУ НЭТИ]
- Пользователь: НГУ
- Бот: Вуз: НГУ [Кнопка: Добавить конкурс]
- Клик
- Бот: Введите ссылку на интересующий вас конкурс
- Пользователь: https://abiturient.nsu.ru/bachelor?faculty=8&direction=6&condition=10&type=0

# Перспективы развития

-   **Улучшение парсеров ВУЗов:** Постоянное обновление и оптимизация алгоритмов парсинга для более точного и быстрого отслеживания позиций в конкурсных списках, а также для повышения устойчивости к изменениям на сайтах ВУЗов.
-   **Расширение списка ВУЗов:** Добавление поддержки новых высших учебных заведений для охвата большей аудитории абитуриентов.
